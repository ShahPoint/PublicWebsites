AutoSpeichern

Haben Sie jemals versehentlich wichtige Daten verloren, nachdem Sie die Zurück-Taste in Ihrem Browser gedrückt haben?

AmbuSoft möchte, dass Sie beim Ausfüllen eines Berichts unbesorgt sind. Deshalb werden unsere Daten automatisch gespeichert, wenn Sie durch das Formular navigieren. So verlieren Sie nie wichtige Daten (auch wenn Sie versehentlich die Zurück-Taste in Ihrem Browser drücken).
