Erzählung Bildner

Erstellen einer Erzählung kann der zeitaufwändigste Teil beim Ausfüllen einer PCR sein.

Wir haben diesen Prozess durch die Verwendung von Vorlagen viel einfacher gemacht! Benutzer können aus einer Reihe von vom Administrator generierten Vorlagen auswählen. Der ePCR durchsucht sein Formular, erfasst die benötigten Daten und füllt den Rest der Vorlage für Sie aus.
